# UI Design Specification for 2min.today ## Core Aesthetic - **Modern Brutalist + Futuristic**, heavily influenced by Apple’s latest UI language - Extremely minimalist and information-dense - Reference vibe: legiblenews.com (clean, typographic) meets early-Google simplicity — but sharper, zero ugliness, zero bloat - **Zero images**. Zero decorative elements. Zero colors or chrome. Pure signal. ## Typography (Heavy Emphasis) - Primary fonts: **Geist / Inter** with variable font weights for instant hierarchy - Monospace: **SF Mono** — reserved exclusively for source citations, links, and disclaimers - Bold one-line section headers for scannability ## Layout & Structure - Rigid grid system only - Single-pixel dividers for all separation - Fixed daily format with exactly **5 core sections**: - **WORLD** • One-line topic header - **TECH** • One-line topic header - **ECONOMY** • One-line topic header - **SCIENCE** • One-line topic header - **CULTURE** • One-line topic header - Rare **Emerging** section only for true outliers (<5 % of content) - Designed to be scanned in <3 seconds and fully read in ~2 minutes ## Content Presentation - High-density per cluster: 3 crisp bullets + 1 “Why it Matters” sentence - All summaries grounded with visible (but unobtrusive) sources in SF Mono - Predictable, repeatable structure — no visual noise
